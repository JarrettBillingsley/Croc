void maybeGC (MDVM* vm);
Runs the garbage collector of the VM if necessary.

This will perform a garbage collection only if a sufficient number of objects have been allocated since the last collection.

Params:
MDVM* vm The VM whose garbage is to be collected.

void gc (MDVM* vm);
Runs the VM's garbage collector unconditionally.

Params:
MDVM* vm The VM whose garbage is to be collected.

uint bytesAllocated (MDVM* vm);
Find out how many bytes of memory the given VM has allocated.
