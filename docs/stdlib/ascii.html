module ascii

This library provides string manipulation and character classification functions which are restricted to the ASCII subset of Unicode. Croc's strings are Unicode, but full Unicode implementations of the functions in this library would impose a very weighty dependency on a Unicode library such as ICU. As such, this library has been provided as a lightweight alternative, useful for quick programs and situations where perfect multilingual string support is not needed.

Note that these functions (except for isAscii) will only work on ASCII strings. If passed strings which contain codepoints above U+00007F, they will throw an exception.

global function isAscii(val: string)

Returns:

a bool of whether val is an ASCII string (that is, all its codepoints are below U+000080).

global function icompare(str1: string, str2: string)

Compares two ASCII strings in a case-insensitive manner.

This function treats lower- and uppercase ASCII letters as comparing equal. For instance, "foo", "Foo", and "FOO" will all compare equal.

Returns:

a negative int if str1 compares before str2, a positive int if str1 compares after str2, and 0 if they compare equal.

Throws:
ValueError

if either string is not ASCII.

global function ifind(str: string, sub: string, start: int = 0)

Searches for an occurence of sub in this, but searches in a case-insensitive manner.

The search starts from start (which defaults to the first character) and goes right. If sub is found, this function returns the integer index of the occurrence in the string, with 0 meaning the first character. Otherwise, if sub cannot be found, #this is returned.

start can be negative, in which case it's treated as an index from the end of the string.

Throws:
ValueError

if either str or sub are not ASCII.

BoundsError

if start is invalid.

global function irfind(str: string, sub: string, start: int = #str - 1)

Reverse case-insensitive find. Works similarly to ifind, but the search starts with the character at start (which defaults to the last character) and goes left.

If sub is found, this function returns the integer index of the occurrence in the string, with 0 meaning the first character. Otherwise, if sub cannot be found, #this is returned.

start can be negative, in which case it's treated as an index from the end of the string.

Throws:
ValueError

if either str or sub are not ASCII.

BoundsError

if start is invalid.

global function isAlpha(c: string, idx: int = 0)

global function isDigit(c: string, idx: int = 0)
global function isHexDigit(c: string, idx: int = 0)
global function isCtrl(c: string, idx: int = 0)
global function isPunct(c: string, idx: int = 0)
global function isSpace(c: string, idx: int = 0)
global function isGraph(c: string, idx: int = 0)
global function isPrint(c: string, idx: int = 0)

These functions all work the same way: they classify a character in a string.

  • isAlpha tests if the character is a lower- or upper-case letter.

  • isAlnum tests if the character is a lower- or upper-case letter or a digit (0-9).

  • isLower tests if the character is a lower-case letter.

  • isUpper tests if the character is an upper-case letter.

  • isDigit tests if the character is a digit (0-9).

  • isHexDigit tests if the character is a hexadecimal digit (0-9, a-f, A-F).

  • isCtrl tests if the character is a control character (less than ASCII 32, or ASCII 127).

  • isPunct tests if the character is a punctuation mark.

  • isSpace tests if the character is whitespace (space, \t, \f, \v, \r, or \n).

  • isGraph tests if the character has a graphical representation (not a control character, and not space).

  • isPrint tests if the character is printable (same as isGraph, but includes space; inversion of isCtrl).

Params:
c

is the ASCII string to look in.

idx

is the index of the character in c to text. Can be negative.

Returns:

true if c[idx] is in that class of character; false otherwise.

Throws:
ValueError

if c is not ASCII, or if #c == 0.

BoundsError

if idx is invalid.
