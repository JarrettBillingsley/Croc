module doctools.console

This module defines a means of outputting docs to the console (or any console-like stream).

global function help(obj, child: string = null)

Get help on a program object and print it to the standard output stream. This function is useful for Croc CLIs.

If obj is a string, it must name a Croc type, and child must also be a string. In this case it looks up the docs for the metamethod named child for the type named obj.

If obj is not a string, and child is null, the docs for obj are printed.

If obj is not a string, and child is not null, the docs for the child named child inside obj are printed.

class BasicConsoleOutputter : DocOutputter

A simple implementation of the doctools.output.DocOutputter class which outputs docs to the console (by default, but can be redirected).

No special terminal features are assumed; no ANSI escapes or anything are used. Furthermore the text is wrapped to a user-settable maximum line length to avoid ugliness.

Links are not translated by this outputter. Only the text within link spans is output.

BasicConsoleOutputter.this(lineEnd: int = 80, output = console.stdout)

Params:
lineEnd

is the number of characters to which line will be wrapped. This simply calls setLineLength with this parameter.

output

is the stream to which the output will be written. The only method that will be called on this is write with string parameters. Defaults to console.stdout.

function BasicConsoleOutputter.setLineLength(lineEnd: int)

Set the number of characters to which each line will be wrapped.

Params:
lineEnd

is the number of characters. The length will be clamped to a minimum of 40 characters; anything less than this will simply set the length to 40.
