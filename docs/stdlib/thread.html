global function halt(t: thread = null)

Halt a thread of execution.

Params:
t

is the thread to halt. If you don't pass anything for this, the current thread will be halted. If t is not running, it will have a pending halt placed on it; that is, the next time it is resumed, it will halt immediately.

namespace thread

function thread.reset(newFunc: function = null)

Resets a dead thread to the initial state.

Params:
newFunc

is an optional function that will replace the thread's old main function. If you pass nothing for this, the thread will just use the same main function that it was created with.

function thread.state()

Returns:

the state of this thread as one of the following strings:

  • "initial": the thread has not yet been called.

  • "running": the thread is currently running. Only the current thread is in this state.

  • "waiting": the thread resumed another thread and is waiting for that thread to yield.

  • "suspended": the thread yielded and is waiting to be resumed.

  • "dead": the thread exited its main function. The only thing you can do with a dead thread is call reset on it.
