module env

This small module allows you to get and set process environment variables.

global function getVar(name: string, defaultVal: string = null)

Get the value of the environment variable named name.

Params:
name

is the name of the environment variable to retrieve.

defaultVal

is an optional value that will be returned in the case that there is no environment variable named name.

Returns:

the value of the environment variable, if any. If no variable named name exists, it will return defaultVal instead (which defaults to null).

global function getAllVars()

Returns:

a table containing all variables in this process's environment. The keys are the names and the values are... the values.

global function setVar(name: string, val: string = null)

Set or unset a variable in this process's environment.

Params:
name

is the name of the variable to set or unset.

val

is the value to set the variable to. If you set the variable to the empty string or null, the variable will be unset (removed from the process's environment).
