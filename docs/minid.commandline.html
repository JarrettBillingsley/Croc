struct CommandLine ;
This struct encapsulates a MiniD command-line interpreter. This is the CLI that MDCL uses, so you can actually embed an MDCL-like interpreter in your application by using this struct.

This struct installs a signal handler that catches Ctrl+C (SIGINT) signals. It restores the old signal handler when it exits.

static CommandLine opCall (Print output, InputStream input);
Construct an instance of the CLI.

Params:
Print output The Print object to which output will be sent.
InputStream input The InputStream from which user input will be gathered.

Returns:
An initialized instance of CommandLine.

static CommandLine opCall (OutputStream output, InputStream input);
Same as above, but takes a raw OutputStream for the output instead of a Print instance.

static CommandLine opCall ();
Constructs a default instance of CommandLine which uses stdin for the input and stdout for the output.

void run (MDThread* t, char[][] args = null);
After you've created an instance of this struct, call this method on it. The arguments are the arguments that would be passed to MDCL, including the program name as item 0. This function returns when the user exits the interpreter (by using the exit() function, usually).

Params:
MDThread* t The thread to use for this CLI.
char[][] args The arguments to the interpreter. Defaults to null, in which case no arguments will be passed.
