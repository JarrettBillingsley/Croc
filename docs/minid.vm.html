MDThread* mainThread (MDVM* vm);
Gets the main thread object of the VM.

MDThread* currentThread (MDVM* vm);
Gets the current thread object of the VM, that is, which thread is currently in the running state. If no threads are in the running state, returns the main thread.

uint bytesAllocated (MDVM* vm);
Find out how many bytes of memory the given VM has allocated.
